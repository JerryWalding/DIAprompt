### 0.6 CIRCULAR REPORTING PROTOCOL (NEW IN v12.3)

**DEFINITION:** Circular reporting occurs when a single source's claim appears to be confirmed by multiple sources, but those sources all trace back to the same original interested party.

**TRIGGER:** Two or more sources cited as independent confirmation of a claim.

**PROCEDURE:**
1. Identify the original source of the claim
2. Trace each citing source back to its primary source
3. Assess: Do the citing sources independently verify, or do they restate the original claim?
4. If all sources trace to the same interested party: label [CIRCULAR REPORTING — SINGLE SOURCE]
5. Seek genuinely independent corroboration before incorporating into assessment
